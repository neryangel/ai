# AI Pipeline Editor

A visual node-based interface for creating AI workflows.

## Features
- Drag & drop nodes onto an infinite canvas
- Connect nodes to create AI pipelines
- Configure AI parameters (OpenAI, Anthropic, Google)
- Execute pipelines and view results

## Tech Stack
- React + TypeScript + Vite
- React Flow
- Tailwind CSS
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```
